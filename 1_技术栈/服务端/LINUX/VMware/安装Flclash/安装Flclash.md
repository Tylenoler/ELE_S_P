# Linux版本
- 进入Releases，选择[deb文件](https://github.com/chen08209/FlClash/releases/download/v0.8.72/FlClash-0.8.72-linux-amd64.deb)下载
![[Pasted image 20250122185755.png]]
- 下载完成后在Linux界面双击打开
![[Pasted image 20250122185839.png]]
- 安装
- 如果可以正常打开，则可以开始正常使用，如不行，进行下一步。
- 执行以下命令：
```
 sudo apt-get install appindicator3-0.1 libappindicator3-dev
 sudo apt-get install keybinder-3.0
```
- 执行完毕后可以打开Flclash。
