# 接口定义1

| 序号  |     标号     |      引脚说明       |
| :-: | :--------: | :-------------: |
|  1  |    VCC     |  电源输入（3.3v~5v）  |
|  2  |    GND     |      电源地线       |
|  3  |     CS     |      LCD片选      |
|  4  |   RESET    |      LCD复位      |
|  5  |     DC     |  LCD总线命令/数据选择   |
|  6  | SDI(MOSI)  | LCD SPI显示总线数据输入 |
|  7  |    SCK     | LCD SPI显示总线时钟信号 |
|  8  |    LED     | LCD背光控制（高电平点亮）  |
|  9  | SDO(MISO)  | LCD SPI显示总线数据输出 |
| 10  |   T_CLK    |  触摸面板SPI总线时钟信号  |
| 11  |    T_CS    |   触摸面板SPI总线片选   |
| 12  |   T_DIN    |  触摸面板SPI总线数据输入  |
| 13  |    T-DO    |  触摸面板SPI总线数据输出  |
| 14  | T_IRQ(PEN) |   触摸面板中断IRQ信号   |
