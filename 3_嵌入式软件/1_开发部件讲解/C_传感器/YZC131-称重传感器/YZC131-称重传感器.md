# 二·引脚定义

**驱动电压：5V~10V**

| 颜色  | 作用     |
| --- | ------ |
| 红色  | 电源正极   |
| 黑色  | 电源负极   |
| 绿色  | 输出信号正极 |
| 白色  | 输出信号负极 |
传感器一端通过螺丝孔固定另一端保持悬空状态，按标签指示方向施加力
**一定注意：不要直接按压白色覆胶部分，以免破坏本传感器**
